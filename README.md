# TransformoDocs

A modern, intuitive document management system for efficient file organization and collaboration.

## Features

- **Intuitive File Management**
  - Drag-and-drop file upload
  - Create, delete, and rename folders
  - Cut, copy, and paste functionality
  - Multi-select with checkboxes
  - Bulk operations support

- **Smart Organization**
  - Hierarchical folder structure
  - Breadcrumb navigation
  - Search by filename or content
  - Filter and sort options

- **User Experience**
  - Dark/Light mode toggle
  - Responsive design
  - File preview support
  - Local storage persistence

## Quick Start

1. Clone repository:
```bash
git clone https://github.com/yourusername/transformo-docs.git
```

2. Install dependencies:
```bash
npm install
```

3. Run locally:
```bash
npm start
```

## System Requirements

- Modern web browser with JavaScript enabled
- Local storage access
- Minimum screen resolution: 1024x768

## Technical Stack

- HTML5/CSS3
- Vanilla JavaScript
- Local Storage API
- SVG for graphics

## File Operations

- Upload: Support for multiple file types
- Download: Direct file download
- Cut/Copy/Paste: Move files between folders
- Delete: Move to recycle bin
- Search: Real-time content/filename search
